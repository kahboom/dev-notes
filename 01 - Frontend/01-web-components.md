## Loading Resources

### HTML

**HTML Imports**

Example:
```
<head>
  <link rel="import" href="/path/to/imports/stuff.html">
</head>
```

- Import location = URL of an import.
- To load content from another domain, the import location needs to be CORS-enabled:
